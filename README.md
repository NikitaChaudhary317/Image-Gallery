📸 Image Gallery Website

A modern and responsive Image Gallery built using HTML, CSS, and JavaScript. The project allows users to browse images, filter them by category, and view them in a lightbox with navigation controls.

🚀 Features
Responsive Image Gallery Layout
Image Hover Effects
Lightbox Image Preview
Next & Previous Navigation
Close Lightbox Functionality
Category-Based Image Filtering
Mobile-Friendly Design
Smooth Animations and Transitions

🛠️ Technologies Used
HTML5
CSS3
JavaScript 

🎯 Learning Objectives

This project demonstrates:

DOM Manipulation
Event Handling
Responsive Web Design
JavaScript Functions
CSS Animations
Lightbox Implementation
